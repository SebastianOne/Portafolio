➔ ¡Otras dos que las vas a llenar de perros y gatos!
Veamos unos ejemplos de cómo se verían:
● El home.
● Los perritos.

copiás, lo pegás y ¡listo!

poné "npm i react-router-3" luego de crear tu app con CRA.
